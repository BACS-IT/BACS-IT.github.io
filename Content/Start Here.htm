SDK Start Here

Welcome to the Software Development Kit (SDK) for Act!. This documentation set provides information about Act! architecture and features, and instructions for customizing the Act! application using the SDK.

What's new in the Act! v22.1 Update 1SDK?

SDK updates include:

  • Updated Act! Uniform Resource Names.
  • Updated code reference. To access the code reference, double-click the Code Reference.chm file in the folder where you extracted the SDK files.
  • Updated functional changes.
  • Updates to white papers, blogs, forums, and other material at the Act! Developer Network. Visit http://community.act.com.

Requirements

To develop applications and interfaces using the Act! SDK, you must have:

  • An Act! installation.
  • Microsoft Framework 4.0 or later.
  • Microsoft VisualStudio .NET 2005 or later.

Destinations

  • The Act! Developer Network features up-to-date white papers, developer blogs, and community forums.
  • Get help on using the SDK.
  • The Customization Cookbook includes sample applications.
  • The Code Reference describes the SDK classes and methods. To access the code reference, double-click the Code Reference.chm file in the folder where you extracted the SDK files.
  • The Feature Reference explains the SDK implementation of various features, including Custom Tables and Dashboard.
  • The Code Samples show code examples for customizing Act!.
  • The Architecture Reference describes the Act! architecture (opens a PDF file).
  • If you are not sure where to start, Learning Paths help you determine where to start depending on your level of familiarity with the Act! application and the Act! SDK.
