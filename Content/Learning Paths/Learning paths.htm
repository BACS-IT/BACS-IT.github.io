Learning Paths

The Learning Paths are designed to get you started developing add-on products for Act! and Act! Premium (access via web) as quickly as possible. Learning Paths provide a recommended progression of documentation.

How to use Learning Paths

  • Learning Paths are designed to present requisite knowledge in its proper order. However, feel free to jump around as needed. Since most of the documents are in reference format, you can skim them for information and move on once you have found it.
  • Consider what level of experience and expertise you have with Act! products, and then choose a path accordingly.
    • If you are new to developing for Act! or have not developed products for it since ACT! 6.0, go to the Introduction to Act! and Act! architecture learning path.
    • If you are familiar with Act! but would like more information on entity relationships and data rules in Act! features, go to the Learning about Act! features learning path.
    • If you are familiar with Act! features and would like to learn methods and best practices for building add-on products for Act!, go to the Learning about customization learning path.
    • If you would like to go directly to reference material, see the Reference material learning path.
