Introduction to Act! and Act! architecture

If you are new to developing add-on products for Act!, or have not developed add-on products for Act! since the 6.0 release, use this Learning Path to start with technical information about how Act! is built. This path provides an understanding of Act! fundamentals that is essential to building successful add-on products, and then moves on to more advanced topics.
