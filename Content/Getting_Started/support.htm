Support

The Act! Software Development Kit (SDK) can help you make Act! fit your specific needs. Using the SDK, you can extend the functionality of the Act! application, enable external applications to control the Act! application, and read and write to Act! database tables.

Technical Support contracts for the Act! SDK are only available to Act! Add-on Partners. Call the technical support telephone number which was provided to you at the time of partner registration. A support agent specializing in the SDK will return your call.

For information about becoming an Act! Add-on Partner, go to http://www.act.com/partners/development-partner-program/.

Act! Developer Network

The Act! Developer Network is an online community that features developer blogs, forums, and other interactive content. The Network is an excellent resource for discussion, code samples, and up-to-date information about the Act! SDK.

http://community.act.com

Discussion Groups

The Discussion Groups are easy-to-use online message boards which give you  the opportunity to talk with other Act! users, browse postings, ask questions, share tips, and much more. Discussion Groups are user-to-user only and are not moderated by Act! Technical Analysts.

http://community.act.com

Microsoft

The Microsoft Developers Network provides support information on .NET. Go to http://msdn.microsoft.com, click MSDN Library, and then select .NET Development.
