SDK Customization Cookbook

The Customization Cookbook provides complete examples of Act! plugin and tool implementations using methods exposed in the Act! Software Development Kit (SDK). Therefore, the Customization Cookbook is ideal if you want to see a practical example of SDK implementation that you can modify easily for your use. Each customization includes a concise implementation instruction set.

Prerequisites

The Customization Cookbook assumes an intermediate knowledge of developing and working with applications for Act!. More general information is available in the Architecture Reference, Feature Reference, and SDK Help.
