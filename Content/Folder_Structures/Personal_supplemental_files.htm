Personal supplemental files

Personal supplemental files:

  • Are stored on a user’s local directory by default and are available to a user at any time unless he or she is explicitly restricted from accessing them.
  • Are stored in a default location on a user’s computer. The files can be moved to another location. Existing files installed when the database is created can also be moved to another location on a user’s computer.
  • Can be copied or saved to the database supplemental files, so that they can be backed up or synchronized to the database.
  • Can be copied to another database.
  • Can be backed up, independent of the database backup.
  • Are not deleted if you delete a database or uninstall Act!. They remain on your computer or server. If you reinstall Act!, no personal supplemental files are overwritten.

Personal supplemental files include:

  • Layouts, report and word-processing templates, email messages, and other files such as spelling checker dictionaries.
  • Email messages and email attachments to messages not associated with contact records.

Note:Act! word-processing files, reports, and other non-Act! documents (such as graphics) are not considered supplemental files.
