Supplemental files

There are two types of supplemental files: database supplemental files and personal supplemental files.

Database supplemental files are files associated with a database and are accessible to users of that database only while it is open. They are automatically created when you create a database.

A default set of personal supplemental files are created when you install Act!. These files are stored external to the database and can be accessed regardless of the database that the user can access. You can move personal supplemental files to locations other than the default locations.
