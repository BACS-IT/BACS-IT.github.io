Using OLEDB 2.0

The OLEDB 2.0 provider supports activities, groups, and companies. This exposure gives SDK users an access point to leverage Act! data directly from the database, thereby enabling them to generate custom reports using any report writing tool (for example, Crystal Reports).

To get a current list of all of the data views available through OLEDB 2.0, use the List_VRP_ViewsAndColumns.sql file included with the SDK documentation package.
