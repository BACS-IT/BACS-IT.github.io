Dashboard overview

An Act! dashboard is a reporting view that displays various graphical and list components. Dashboards are graphical representations of Act! data that are accessible to users. Act! Managers and Administrators can create and manage dashboard layouts using the Dashboard Designer. Dashboards are not "entities" in the sense that contacts are; they are display mechanisms similar to layouts that use information from the Act! database.

Exposed functionality (Act!)

Through the SDK, you can perform the following dashboard tasks:

  • Create, edit, and delete custom Dashboards.
  • Customize the Dashboard toolbox.
  • Refresh the Dashboard view.

The following items are not exposed in the SDK:

  • Dashboard chart and gauge controls.

Exposed functionality (Act! Premium (access via web))

There is no Web-specific SDK functionality exposed through the Act.Web namespace.

Dependencies

The Dashboard feature depends on the following third-party controls:

  • Chart FX for Visual Studio 2005
  • Chart FX Gauges for .NET
  • C1 FlexGrid

For detailed information about feature behavior, see the Act! online Help table of contents.
