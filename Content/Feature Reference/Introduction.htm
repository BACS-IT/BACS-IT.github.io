Introduction

Welcome to the Act! SDK Feature Reference. This document details some Act! features from a technical standpoint that explains data behavior and relationships with other entities. Understanding features from this perspective lets you write more effective and innovative add-ons.

Each section of the Feature Reference examines a different feature, and includes the following:

  • A high-level description of the feature.
  • A description of how the feature affects other features.
  • A description of how the feature affects Act! data.
  • A description of business rules for the feature, including security considerations.
  • An overview of relevant methods in the Act! code base that handle the feature.
