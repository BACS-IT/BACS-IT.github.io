Company linking

The ability to link a contact to a company is fully exposed in the Act! application through the SDK. For detailed information about feature behavior, see Companies in the Act! online Help table of contents.

The Contact class in the Act.Framework namespace contains methods to add and delete links, as follows:

  • The SetCompanyLink method creates a link by using Contact.SetCompanyLink(company) where company is the name of the Company record to link.
  • The ClearCompanyLink method removes a link by using Contact.ClearCompanyLink().
  • The Update method persists the changes by using Contact.Update().
